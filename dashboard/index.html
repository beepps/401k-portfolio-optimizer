Methodology & Data Sources
How this analysis was built and verified
📊 Backtest Engine
250 unique weight combinations generated via structured grid search (aggressive/balanced/conservative) + Dirichlet Monte Carlo sampling. Each blend tested against 183 months of actual ETF price data (Jan 2011–Apr 2026) with monthly rebalancing assumed.
🎯 Fund Proxies
Since your 401K funds (ZBGRIT, WMTX8, etc.) are collective investment trusts or private funds, publicly-traded benchmark ETFs were used: iShares Russell 1000 (IWB) for ZBGRIT, SPDR S&P 500 (SPY) for WMTX8, iShares MSCI EAFE (EFA) for ZDEVLT, AGG for bond funds, etc. Actual fund performance may differ slightly from proxy.
📐 Metrics Used
Sharpe Ratio: Risk-adjusted return above 2% risk-free rate.
Sortino Ratio: Only penalizes downside volatility.
Calmar Ratio: Annual return / max drawdown.
Max Drawdown: Largest peak-to-trough decline.
📰 Premium Verification
Asset class annual returns fact-checked against: Statista ICI Retirement Data, Russell 1000 Index historical returns, Bloomberg Aggregate Bond Index annual returns, BlackRock International Index Fund (BlackRock), PitchBook fund profiles, and CB Insights wealth management analysis.
